# Web Waifu 4

Standalone Twitch-first web waifu stream overlay.

- Frontend: React, Vite, Three.js, VRM, browser/local assets.
- Backend: Node server for OpenAI Responses, tools, Twitch/overlay bridges, and remote TTS proxying.
- Build: `npm run build`.
- Deploy: build and run the frontend/static server plus the AI proxy on the VPS.
