Local Speed Harness

LM Studio stream in, Piper queue out.

This demo streams assistant text from LM Studio, slices it into speakable chunks, sends those chunks through the local `Riko` Piper voice, and plays them back in order. The point is timing: first token, first ready audio, queue depth, and how quickly the browser keeps up once the model is warm.
